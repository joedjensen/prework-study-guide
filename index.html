HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Included in the head are title, style, base, link, meta, and script.
  • Section elements are semantic elements that define thematic sections.

CSS

  • Inline CSS is written directly in HTML. Styles are defined per element. Not flexible.
  • Internal CSS allows greater flexibility by including a style sheet in the style element of the HTML.
  • External CSS links an external style sheet
  • Styles are declared in key value pairs and can be assigned to elements or classes
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • * is a wildcard selector, applying the style to all elements.
  • Class selectors are preceded with a "."

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Use script to link the JavaScript scripts to the HTML.
  • Use eventHandlers to make page truly dynamic.
  • Declare variables using let. Variable names are case sensitive.
  • Variable types are string, number, bool, array, object.
  • Loosely and dynamically typed. Any variable can be reassigned to any type.
  • if (condition) {code to execute}
  • function functionName(parameters) {code to execute}
  • function(params) {code to execute} or (params) => {code to execute} for lambdas.
  • use for loops to iterate through arrays
